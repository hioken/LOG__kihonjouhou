# 基礎計算
## 基数変換
- 2 => 10の変換方法
```
桁の重みを足していく
```
- 2 => 8, 16の変換方法
```
3/4桁ごとに、0 ~ 7/Fに治す
```
- 10 => 2の変換方法
```
2で割っていって、余りをメモしてく
```
- 16 => 2の変換方法
```
1桁ずつ、2進数に変換していく、必ず4桁になるように0で埋める
```
- 10 => 16の変換方法
```
16で割ってく以下略
```
- 2に変換したら無限小数になる10進数を書いて
```
0.2
```
